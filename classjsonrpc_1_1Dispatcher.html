Detailed Description

A class responsible for dispatching JSON-RPC requests to the appropriate handlers.

The Dispatcher class enables the registration and execution of method call and notification handlers for JSON-RPC requests. Handlers can be registered using std::function, allowing for flexible and modern C++ usage. The Dispatcher can operate in single-threaded or multi-threaded mode, providing scalability for handling batch requests.
